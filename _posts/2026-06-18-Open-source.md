Open source tools are ones where the source code is freely available and licensed in a manner that permits modification. Open-source tools are often as good or even better than proprietary ones (which are privately owned, or controlled by one particular organization).

Below is a list of open source tools that can be used for education, work, or other pursuits. 

Free dictionary, Wiktionary: https://www.wiktionary.org/
Computer operating system, Ubuntu: https://ubuntu.com/ (there are other distributions)
Programming distribution for python, anaconda: https://www.anaconda.com/download (an easy starting point is anaconda navigator)
Free AI image upscaler: https://free.upscaler.video/
Packing and unpacking files, 7-zip: https://www.7-zip.org/
Molecular dynamics, GROMACS: https://www.gromacs.org/
Hybrid tower-defense sandbox factory game, Mindustry: https://anuke.itch.io/mindustry

There are a number of good tools that are free but not open source. 

Scientific calculator and graph plotter, desmos: https://www.desmos.com/scientific

If you have any more ideas let me know and I will add them.
